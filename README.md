# PCDoor 
Aplicativo mobile desenvolvido originalmente como Trabalho de Conclusão de Curso (TCC) e posteriormente refatorado e modernizado.

## Sobre o projeto
O PCDoor é um aplicativo de avaliação de acessibilidade de estabelecimentos. A proposta é permitir que usuários avaliem e consultem o nível de acessibilidade de diferentes locais, promovendo mais informação e inclusão.

## Funcionalidades
- Cadastro e login de dois tipos de conta: Usuário e Estabelecimento
- Listagem de estabelecimentos cadastrados com busca por nome ou tipo
- Avaliação de itens de acessibilidade por categoria (Motora, Visual e Auditiva)
- Comentários nas avaliações
- Visualização das notas de acessibilidade por categoria e por item
- Painel do estabelecimento com médias de avaliações e comentários recebidos

## Tecnologias utilizadas
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
